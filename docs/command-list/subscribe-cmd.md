# 授权相关命令

## 赞助

查看 AXBot 的赞助入口说明。目前该命令仅在群聊中可用。

```bash title="命令格式"
@AXBot /赞助
```

更多赞助说明请查看 [赞助与合作](../sponsor.md)。

## 激活

!!! success "本命令可在私聊中使用"

激活 [赞助](../sponsor.md) 或其他活动获得的小礼物。

```bash title="命令格式"
@AXBot /激活 <礼物码>
```

!!! warning "建议在私聊中使用"

    礼物码只能使用一次。虽然当前代码也能在群聊中处理该命令，但为了避免泄露礼物码，建议只在私聊中激活。

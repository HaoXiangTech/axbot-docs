# QQ用户/QQ群相关

## 群状态

查看本群状态，包括群 OpenID、认证管理员数量、群特权、群雷币、战绩图模板和近 7 天活跃度。

!!! tip "关于 group_openid"
    AXBot 无法得知 QQ 群的群号，仅能通过 `group_openid` 定位群。如果你的问题和具体的群相关，请提供 `group_openid`。

```bash title="命令格式"
@AXBot /群状态
```

## 个人状态

!!! success "本命令可在私聊中使用"

查看个人状态，包括用户 OpenID、角色、账号状态、亲密度和剩余雷币。

!!! tip "关于 user_openid"
    AXBot 无法得知 QQ 用户的 QQ 号，仅能通过 `user_openid` 定位用户。如果你的问题和具体用户相关，请提供 `user_openid`。

```bash title="命令格式"
@AXBot /个人状态

# 兼容旧叫法
@AXBot /用户状态
```

## 个人设置

!!! success "本命令可在私聊中使用"

管理个人偏好。目前支持切换 AXBot 的回复样式。

```bash title="命令格式"
@AXBot /个人设置

@AXBot /个人设置 切换回复样式
```

回复样式有两种：

- 带按钮版：默认样式，使用富文本卡片和快捷按钮，适合新版 QQ 客户端或小程序。
- 兼容版：使用纯文本加图片消息，不带按钮，对旧版 QQ 客户端兼容性更好。

## 群管理

管理本群的战绩查询和签到开关。本命令仅群聊可用，且需要先完成群管理员认证。

```bash title="命令格式"
@AXBot /群管理

@AXBot /群管理 开关战绩查询

@AXBot /群管理 开关签到
```

当前可用子命令：

- `开关战绩查询`：切换本群是否允许使用战绩相关查询命令。
- `开关签到`：切换本群是否允许使用签到命令。

!!! question "我明明是管理员，为什么提示我不是？"

    受 QQ 机器人平台能力限制，AXBot 无法直接实时读取群成员身份。你需要在群里点击机器人头像，找到“接受机器人的推送”开关，并任意切换一次，AXBot 才能记录你的群管理员身份。

!!! warning "群友被解除管理员后为什么还能用群管理？"

    AXBot 无法实时感知群管理员身份变化。如果出现这种情况，请到反馈群联系开发者处理。

## 雷币日志

!!! success "本命令可在私聊中使用"

查看最近 10 条雷币变动记录。私聊中查看个人雷币日志，群聊中查看本群雷币日志。

```bash title="命令格式"
@AXBot /雷币日志
```

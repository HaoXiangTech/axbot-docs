# 娱乐相关命令

## 签到

!!! success "本命令可在私聊中使用"

每日签到，获得雷币奖励，展示每日气运，并让 AXBot 随机挑选一辆载具作为今日幸运载具。

```bash title="命令格式"
@AXBot /签到
```

签到规则：

- 每次成功签到默认获得 2 雷币。
- 气运值为 100 时，本次签到奖励会提高到 33 雷币。
- 如果所在群启用了有效的群特权，签到奖励会获得额外加成。
- 今日幸运载具仅供娱乐，由 AXBot 在载具列表中随机挑选，不会使用任何玩家的战绩数据作为依据。

!!! note "关于亲密度"

    亲密度目前暂未提供实际功能，签到也不再提高亲密度；历史亲密度数据仍会保留。

## 叫我

!!! success "本命令可在私聊中使用"

让 AXBot 记住你的称呼。后续部分回复会使用这个称呼问候你。

```bash title="命令格式"
@AXBot /叫我 <昵称>
```

昵称最长 16 个字符，并会经过内容审核；不合规的昵称会被拒绝。

# 基础命令

## 帮助

!!! success "本命令可在私聊中使用"

查看 AXBot 的帮助信息、文档入口和常用命令快捷入口。

```shell title="命令格式"
@AXBot /帮助
```

## 问题反馈

显示问题反馈方式；也可以直接在命令后写下要反馈的问题，AXBot 会记录并转交给开发团队。

```shell title="命令格式"
@AXBot /问题反馈

@AXBot /问题反馈 <你遇到的问题>
```

如果问题比较复杂，仍然建议通过 [反馈和建议](../report.md) 中的渠道联系开发者，并附上截图或录屏。

## 系统信息

!!! warning "这是一个隐藏命令，仅超级管理员可用"

用于查看 AXBot 当前版本、CPU、内存和硬盘使用情况。

```shell title="命令格式"
@AXBot /系统信息
```

## 统计

!!! warning "这是一个隐藏命令，仅超级管理员可用"

查看 AXBot 的用户、群聊、消息量和已记录战雷玩家数据等统计信息。

```shell title="命令格式"
@AXBot /统计
```

# 战雷相关命令

## 通用参数写法

战绩相关命令里的“玩家”参数支持三种写法：

- `玩家昵称`：按昵称查询；昵称包含空格时，请用英文双引号或中文引号包裹。
- `**玩家ID`：按战雷玩家 ID 精确查询。
- `..序号`：使用 `/快捷绑定` 绑定列表中的序号。

```bash title="玩家参数示例"
@AXBot /战绩总览 玩家A
@AXBot /战绩总览 "Player With Space"
@AXBot /战绩总览 **123456
@AXBot /战绩总览 ..1
```

!!! note "关于消耗和限速"

    战绩相关查询默认会消耗雷币；如果你是超级管理员、赞助用户、志愿者，或处于特定广告活动时间，部分查询不会消耗雷币。战绩查询存在限速：单用户默认 10 次/分钟，群聊中单群默认 20 次/分钟。

## 快捷绑定

!!! success "本命令可在私聊中使用"

绑定战雷玩家 ID 到自己的快捷列表中，之后可以用 `..1`、`..2` 这样的序号代替昵称或 ID。最多绑定 10 个玩家。

```bash title="命令格式"
@AXBot /快捷绑定

@AXBot /快捷绑定 <玩家昵称>

@AXBot /快捷绑定 "<带空格的玩家昵称>"

@AXBot /快捷绑定 **<玩家ID>

@AXBot /快捷绑定 <玩家昵称或**玩家ID> 解绑
```

## 战绩总览

!!! success "本命令可在私聊中使用"

查看战雷玩家的总体战绩图，包括个人信息和各模式下的主要数据。

```bash title="命令格式"
@AXBot /战绩总览 <玩家> [强制刷新]
```

示例：

```bash
@AXBot /战绩总览 玩家A
@AXBot /战绩总览 ..1
@AXBot /战绩总览 **123456 强制刷新
```

可用参数：

- `强制刷新`：额外消耗雷币，强制刷新该玩家战绩快照。

查询成功后，返回内容会提示下一次自动刷新时间。默认情况下，已缓存的玩家数据会在 7 天后自动刷新。

## 载具战绩详情

!!! success "本命令可在私聊中使用"

查看某个玩家指定载具的详细战绩图。

```bash title="命令格式"
@AXBot /载具战绩详情 <玩家> <载具ID或官方 wiki 链接>
```

示例：

```bash
@AXBot /载具战绩详情 玩家A cn_ztz_99a
@AXBot /载具战绩详情 ..1 cn_ztz_99a
@AXBot /载具战绩详情 **123456 https://wiki.warthunder.com/unit/cn_ztz_99a
```

### 怎么找到载具 ID

- 使用 `/战雷载具数据 搜索 <参数>` 搜索载具。
- 如果你不知道载具 ID，也可以粘贴官方 War Thunder Wiki 的载具页面链接，AXBot 会尝试从链接中识别载具 ID。

## 载具战绩分析

!!! success "本命令可在私聊中使用"

按指定维度分析某个玩家的载具战绩，并生成图表。

```bash title="命令格式"
@AXBot /载具战绩分析 <玩家> <分析维度>
```

示例：

```bash
@AXBot /载具战绩分析 玩家A 历史KD
@AXBot /载具战绩分析 ..1 历史胜率
@AXBot /载具战绩分析 **123456 街机场次
```

支持的分析维度：

- `街机KD`
- `历史KD`
- `全真KD`
- `街机KR`
- `历史KR`
- `全真KR`
- `街机场次`
- `历史场次`
- `全真场次`
- `街机胜率`
- `历史胜率`
- `全真胜率`

## 战雷载具数据

查询战雷载具拆包数据，包括载具名称、国家、类型、等级、权重、价格、经验倍率、奖励倍率等信息。

```bash title="命令格式"
@AXBot /战雷载具数据

@AXBot /战雷载具数据 <载具ID>

@AXBot /战雷载具数据 <模糊载具名称>

@AXBot /战雷载具数据 搜索 <参数1> [参数2] [参数3] ...
```

使用方式：

- 已知载具 ID 时，直接输入 ID，例如 `/战雷载具数据 cn_ztz_99a`。
- 只知道部分名称时，直接输入模糊名称；如果匹配到多个载具，AXBot 会展示第一个结果，并列出其他可能的载具 ID。
- 想明确筛选列表时，使用 `搜索` 子命令。

`搜索` 支持的参数：

- 系别：`美系`、`苏系`、`德系`、`英系`、`日系`、`意系`、`法系`、`瑞系`、`中系`、`以系`
- 类型：`陆军`、`空军`、`海军`
- 历史权重：例如 `1.0`、`1.3`、`1.7`、`12.7`

## 战雷梗图

随机发送一张战雷相关梗图；也可以指定图片标识查看某一张图。

```bash title="命令格式"
@AXBot /战雷梗图

@AXBot /战雷梗图 <图片标识>
```

AXBot 返回随机梗图时，会在消息中附带图片标识。之后可以使用该标识再次查看同一张图。

!!! warning "发现令人不适或不合适的图片"

    图片收集自网络，开发者无法逐张人工筛选。如果你认为某张图片不应出现在公共场合，请按 [反馈和建议](../report.md) 中的方式联系开发者。

    旧版命令目前仍在代码中保留，用于兼容旧战绩图和旧参数行为，但未来可能会继续调整或移除。

### 战绩总览旧

!!! success "本命令可在私聊中使用"

旧版战绩总览图。相比新版 `/战绩总览`，旧版额外支持 `对比历史`、`隐藏信息`、`免费` 等参数。

```bash title="命令格式"
@AXBot /战绩总览旧 <玩家> [参数1] [参数2] ...
```

旧版可用参数：

- `对比历史`：额外消耗雷币，对比上一次记录的数据。
- `隐藏信息`：隐藏个人信息，便于分享。
- `强制刷新`：额外消耗雷币，强制刷新数据。
- `免费`：低清免费模式；不能同时使用 `对比历史` 或 `强制刷新`。

# 赞助与合作

AXBot 作为国内唯一一个服务于战雷玩家的聊天机器人（截止2025年5月），AXBot 有庞大的用户群体。这意味着 AXBot 存在一定的运营成本。特别是随着时间的推移，AXBot 的数据量将会越来越大，运营成本也会越来越高。

这笔费用从原来开发者一个人就能承受的水平，到现在已经是一笔不小的开支。

## 初代机器人的经验和教训

开发者曾经开发过几个类似的机器人（AXBot 一代，WTBot），这几个机器人最终都因为没有收入来源覆盖掉维护的花费而关停服务。

**前车之鉴告诉我们，“用爱发电”的模式并不可持续。**

然而，作为一个希望被国内社区广泛接纳和使用的聊天机器人，AXBot 承诺不会在基础功能上向用户收取费用。

**因此，AXBot 的长期运营离不开个人赞助者的支持，比如赞助；以及和赞助商合作，比如给赞助商打广告。**

## 个人赞助

AXBot 目前面向个人赞助者只支持唯一的赞助方式，即在爱发电平台上发布“赞助包”。赞助者可以以购入“赞助包”的方式来支持 AXBot 的开发和运营。

[跳转到爱发电首页](https://ifdian.net/a/axbot?tab=home){ .md-button .md-button--primary }

需要注意的是，虽然我们十分乐于见到用户赞助 AXBot 机器人，但是我们希望您能注意阅读爱发电页面上的赞助说明，以免产生不必要的误会。

### 对个别开发者的私人赞助

如果你想要对个别开发者进行私人赞助，请自行联系开发者，不要通过爱发电赞助。爱发电的赞助均会根据AXBot开发团队的内部规定进行分配。

### 赞助者特权

赞助者为了广大用户的使用体验奉献了真金白银的支持，因此赞助者将会获得一些普通用户没有的特权。比如：

- 赞助者的问题反馈和功能建议会被优先响应
- 赞助者将会获得额外的雷币（AxBot的积分）奖励

但我们还是需要强调的是， **成为赞助者并不代表着你是 AXBot 的金主，并不能对 AXBot 或者其开发团队指手画脚。AXBot开发团队有权力拒绝任何不合理的要求和索取。**

## 商业合作

AXBot 目前对和赞助商合作持开放态度，欢迎有意向的赞助商和我们联系。

开发者电子邮箱：[axiangcoding@gmail.com](mailto:axiangcoding@gmail.com){ .md-button }

### 赞助商合作

如果您是一家企业，想要通过 AXBot 机器人来宣传您的产品或服务，您可以选择成为 AXBot 的赞助商。

目前 AXBot 可以提供的合作方式有：

- 在玩家战绩图中加入赞助商的广告。
- 在日常签到图中加入赞助商的广告。

### 定制群特权

除了赞助商合作之外，AXBot 也提供面向群聊的定制群特权，适合希望为特定群聊定制专属体验的群主或管理员。

定制群特权以一年为周期，费用一般会大于 300 人民币，具体细节需要根据定制需求进一步沟通确认。定制群特权包括：

- 高度定制战绩图
- 签到特权
- 如群聊无法继续使用，可免费迁移一次到新的群聊中；第二次开始，每次迁移收费 50 人民币

### 群雷币说明

群雷币和定制群特权无关，是独立于定制群特权之外的机制。购买同等数量的群雷币，需要支付个人雷币 1.5 倍的金额。

举例来说，如果某个赞助金额可以获得 1000 个人雷币，那么购买 1000 群雷币时，需要支付该赞助金额的 1.5 倍。

当然，上述合作与定制都需要在遵循腾讯机器人平台运营规范的前提下进行。具体方式和细节请联系开发者，开发者会根据实际需求提供定制化的合作方案。

### 关于数据共享的说明

AXBot 机器人不会向第三方共享我们的用户数据以及我们所使用的游戏内的任何数据。
